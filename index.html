AngularJS Custom Inputs

Custom Checkboxes

Example 1

This example uses default true | false values.

ngModel value: {{cbValue}}
Checkbox toggled: {{changeCounter}} times 

<label for="checkbox1">Label outside directive template</label>

<custom-checkbox checkbox-id="checkbox1"
	model="cbValue"
	label="Check Me"
	>
</custom-checkbox>

<button ng-click="cbValue = !cbValue">Toggle checkbox via JS</button>

Example 2

This example uses ng-true-value and ng-false-value on the generated checkbox to use non-standard values when the checkbox is toggled. Adjust all functions accordingly.

ngModel value: {{cb2Value}} 

<custom-checkbox
	checkbox-id="checkbox2"
	model="cb2Value"
	label="Toggle Me"
	true-value="'truthy'"
	false-value="0"
	>
</custom-checkbox>
