Example 1

This example executes a function when the checkbox is changed (ng-click instead of ng-change necessary because ng-change only responds to direct user interactions). The checkbox uses default true | false values.


ngModel value: {{cbValue}}
Checkbox toggled: {{changeCounter}} times 

<label for="checkbox1" ng-click="changeFn()">Label outside directive template</label>

<custom-checkbox checkbox-id="checkbox1"
	model="cbValue"
	label="Check Me"
	on-click="changeFn()"
	>
</custom-checkbox>

<button ng-click="cbValue = !cbValue; changeFn()">Toggle checkbox via JS</button>

Example 2

This example uses ng-true-value and ng-false-value on the generated checkbox to use non-standard values when the checkbox is toggled. Adjust all functions accordingly.

ngModel value: {{cb2Value}} 

<custom-checkbox
	checkbox-id="checkbox2"
	model="cb2Value"
	label="Toggle Me"

	>
</custom-checkbox>
